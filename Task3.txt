Task3:Explain the use of grid-auto-row and grid-auto-column using code example?
sol : grid-auto-row and grid-auto-column properties specify the height and width of rows
      that are automatically created when there is no explicit row definition and column 
      definition respectively.
      example:- This examples creates a grid container with two columns defined by grid-template-areas
		The first row is displayed  with a height of 50px because we have mentioned grid template
		rows for the first row as 50px and second row are displayed with a height of 200px.
